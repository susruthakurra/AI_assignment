# AI Based Travel Planner

## Overview
This project implements an AI based travel planner using a simple knowledge base. The system recommends travel destinations based on user preferences, budget, and travel interests.

The planner uses predefined knowledge about:
- Tourist places
- Food recommendations
- Budget estimation
- Travel categories

The project demonstrates how Artificial Intelligence systems can use knowledge representation and rule-based recommendations for decision making.

## Features
- Destination recommendation
- Budget-based filtering
- Food recommendations
- Tourist place suggestions
- Personalized travel planning

## Files Included
- knowledge_base.json
- travel_planner.py
- sample_output.txt

## Technologies Used
- Python
- JSON Knowledge Base
- Rule-Based Recommendation System

## Objective
To design an AI based travel planner that reuses existing knowledge bases and provides personalized travel recommendations.
